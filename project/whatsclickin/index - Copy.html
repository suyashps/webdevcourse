Instagram Authentication & Location Search

Authentication

This demo tests Instragram client side authentication. To see the demo, click on the button below and you will be redirected to Instagram authentication page. When this page is displayed, enter your credentials and click login. Instagram will redirect you to this page again and if login is successful you will see an alert message.

The access token received from instagram will be shown in the text box below.
